NIFTY 50 Real-Time Market Data Analysis
A dynamic Power BI project that extracts real-time NIFTY 50 stock market data via web scraping. The report features a live-refresh capability, allowing users to fetch the latest market updates instantly with a single click.

🚀 Project Overview
Monitoring the stock market in real-time is crucial for day traders, analysts, and investors. This project bridges the gap between live web data and interactive analytics. By implementing a direct web scraping pipeline into Power BI, this dashboard transforms raw financial data into actionable, real-time visual insights.

🔑 Key Features
Live Web Scraping: Automatically extracts real-time market data directly from financial web sources via Power Query.

On-Demand Refresh: A single click on the "Refresh" button updates the entire dashboard with the latest market changes.

Comprehensive Metrics: Tracks key financial indicators including Open, High, Low, Current Market Price (CMP), Change (%), and 52-Week High/Low.

Dynamic Visualizations: Highlights top gainers, top losers, and sectoral performances using interactive charts.

🛠️ Tech Stack & Tools
Data Source: Web Scraping (Financial Portals / NSE-related public data)

ETL & Data Transformation: Power Query (M Formula Language)

Analytics & Data Modeling: Power BI Desktop

DAX: Custom measures for dynamic performance tracking (e.g., daily percentage shifts, volatility metrics)

📊 Dashboard Architecture & Insights
The dashboard is structured into three primary areas of analysis:

Market Overview KPI Cards: Instantly view the current state of the NIFTY 50 index, overall market momentum, and advanced-to-decline ratios.

Top Gainers & Losers: A dynamically sorted leaderboard showing the stocks moving the market the most in real-time.

Price vs. 52-Week Range: A visual gauge or scatter analysis showing how close individual NIFTY 50 stocks are to their yearly highs or lows.
